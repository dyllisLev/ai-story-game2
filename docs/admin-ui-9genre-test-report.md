## 5. 발견된 문제점 및 제안

### 5.1 API 키 설정 필요 ⚠️ **[BLOCKER]**
**문제**:
- Play 페이지 진입 시 "API 키를 입력하세요" 메시지 표시
- 실제 스토리 생성 및 플레이 불가

**영향**:
- 9장르 실제 플레이 테스트 불가
- 스토리 품질 평가 불가

**제안**:
1. 테스트용 Gemini API 키 발급 필요
2. 또는 개인 API 키 사용에 대한 가이드라인 제공

### 5.2 테스트 범위 재조정 필요
**현실적 제약**:
- 9개 장르 × 10-15턴 플레이 = 약 2-2.5시간 소요
- API 키 없이 실제 플레이 테스트 불가

**제안하는 접근**:
1. **1단계: UI 기능 테스트** (완료)
   - Editor UI/UX 검증
   - 장르별 설정 기능 확인
   - 프리셋 작동 확인

2. **2단계: 샘플 플레이 테스트** (API 키 필요)
   - 1-2개 장르만 실제 플레이
   - 10-15턴 진행
   - 품질 평가 기준 적용

3. **3단계: 전체 테스트** (선택)
   - 나머지 7-8개 장르 플레이
   - 각 장르별 품질 평가
